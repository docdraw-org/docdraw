# docdraw
Deterministic document standard + local toolchain (DocDraw v1).

## What this repo is
- **Standard**: DocDraw v1 (canonical format), DMP‑1 (Markdown profile), DD‑PDF‑1 (PDF renderer profile)
- **Tooling**: `./bin/docdraw` (validate, normalize, convert, render)
- **Conformance**: golden examples + `examples/golden-manifest.json`
